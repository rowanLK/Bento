bento/screen

Screen object. Screens are convenience modules that are similar to levels/rooms/scenes in games. Tiled Map Editor can be used to design the levels http://www.mapeditor.org/. Note: in Tiled, you must export as json file and leave uncompressed as CSV (for now)
